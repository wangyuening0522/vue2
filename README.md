# vue
## 技术栈
vue2，vue-router,axios
## 亮点
使用图片懒加载，实现登录功能(前置路由守卫)
## 功能
1. 修改个人信息
2. 二维码登录
3. 侧边栏动画
4. 个人页面效果还原
5. 歌单详情页
